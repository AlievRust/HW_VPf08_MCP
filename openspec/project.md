# Проект: notes-mcp

## Назначение

Учебный Python-проект для работы с MCP-сервером заметок и CLI-клиентом на OpenAI Responses API.

## Цели

- показать полный цикл tool calling через MCP;
- хранить заметки локально в SQLite;
- дать понятный интерактивный CLI для обычного пользователя;
- сделать проект простым, воспроизводимым и удобным для учебного разбора.

## Ограничения

- только локальный SQLite;
- только MCP tools, без произвольных REST endpoint'ов;
- все секреты и ключи хранятся только в `.env`;
- запуск должен быть возможен без ручной подготовки базы;
- пользовательский интерфейс и документация должны быть на русском языке.
